    launch_args: []
    env: {}
    capabilities_max:
      fs_read: true
      fs_write: true

## Risks and Mitigations

| Risk | Severity | Mitigation |
|------|----------|------------|
| 配置模型改动导致加载失败 | 高 | schema 测试 + fail-fast 诊断错误 |
| role 解析错误导致 stage 无法执行 | 高 | RoleResolver 单测覆盖 missing role/missing agent/capability overflow |
| API 动作语义切换引发回归 | 中 | handlers 单测覆盖 change_role 路径 |
