# Backend 目录说明

本目录是后端主规范（Authoritative）。

文件：
- `spec-overview.md`：架构总览与保留/剔除决策
- `spec-api-config.md`：真实可用 API 面
- `spec-config-runtime.md`：配置加载层次与环境变量覆盖
- `spec-run-engine.md`：Run 状态机与 action/事件
- `spec-team-leader-layer.md`：Team Leader 会话与 issue 链路
- `spec-github-integration.md`：GitHub webhook 与 slash command 边界
- `spec-agent-drivers.md`：chat/WS/事件驱动契约
- `spec-context-memory.md`：OpenViking 现状边界
